# 乘法小天才 V3.0 需求补充与原型建议

> 配套文件：`需求理解报告_儿童乘法APP.md`
> 目的：补全当前原型（22 页，覆盖 V1.0 + 大部分 V2.0）中 **缺失的 V3.0 功能页面**——多账号切换、应用题专项、数据导出、错题精准讲解、学习时长精细化管控。
> 设计约束：沿用现有设计语言（手机 390 宽竖屏 / 平板 1024 横屏；底部导航 首页｜训练｜错题本｜我的；暖色护眼基调；童声听题、数字键盘、卡片+开关+滑块组件）。

---

## 一、V3.0 缺口确认（基于原型真实数据）

| PRD 规划 V3.0 功能 | 原型现状 | 缺口结论 |
| --- | --- | --- |
| 乘法应用题专项（L5） | 题型含「简单图文应用题」字样，但**无独立页面/答题页** | ❌ 需新增 |
| 错题精准讲解 | 仅「记忆小提示」轻量版（如"七八五十六…"） | ❌ 需升级为分步讲解 |
| 多账号切换（多孩子） | 个人中心仅单用户"小明同学"，无切换入口 | ❌ 需新增 |
| 学习时长精细化管控 | 仅"每日总时长 + 禁用时段" | ⚠️ 需补充分模块/休息提醒 |
| 数据导出 | 成长报告页无导出/分享入口 | ❌ 需新增 |

> 核查关键词命中：原型中"应用题/多账号/导出/数据导出/精准讲解/精细化"均**未出现**；"切换"仅命中护眼夜间切换，"护眼"为现有功能。结论可靠。

## 三、现有页面增强建议（非新增，原地升级）

### 增强 1：护眼设置页 → 增加「学习时长精细化管控」

在现有 `手机-护眼设置页` 基础上新增区块（不影响现有护眼功能）：

| 新增项 | 设计建议 |
| --- | --- |
| **分模块时长** | 「启蒙入门 / 专项进阶 / 口诀游戏」各自可设上限（如 启蒙 15 分钟、进阶 20 分钟），总和不超过每日总时长 |
| **休息提醒** | 开关 + 间隔设置（每 20 分钟 / 30 分钟提醒休息，默认开） |
| **多禁用时段** | 现有仅单一段（21:00-07:00），升级为可添加多个时段（如 午休 12:00-14:00） |
| **文案** | 「分模块时长限制」「休息提醒：保护视力，学一会儿歇一歇」「添加禁用时段」 |

### 增强 2：进阶训练-难度选择页 → 开放 L5 入口

将现有 L3「提速训练 🔒」之后新增 `L5 进阶拔高`（V3 解锁），点击进入应用题专项（页面 B）。

## 五、一致性检查 & 落地建议

1. **命名规范**：新增手机页统一 `手机-XXX`，平板页 `平板-XXX`，与原 22 页一致。
2. **导航一致**：所有新页沿用底部 4 Tab（首页/训练/错题本/我的）；账号切换为"我的"层入口，不破坏底部导航。
3. **组件复用**：数字键盘、童声听题(📢)、卡片、开关、滑块、星级、勋章均复用现有；避免引入新视觉语言。
4. **护眼基调**：背景 `rgba(255,248,240,1)`、圆角 24 顶卡片、橙色主色，与全站统一。
5. **数据隔离**：多账号下，进度/积分/错题/报告必须按 child_id 隔离，切换即换库。
6. **建议落地顺序**：多账号切换（基础，影响数据模型）→ 应用题专项（L5 内容）→ 错题精准讲解（体验升级）→ 精细化管控 → 数据导出（依赖报告体系）。
7. **可验证性**：每个新页均可在 Calicat 中按现有页面结构 clone 出骨架，再填充上述文案与组件。
