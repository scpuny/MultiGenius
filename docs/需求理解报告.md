## 一、产品定位总览

| 维度     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| **名称**   | 儿童乘法入门与进阶训练APP（暂定名）                             |
| **核心赛道** | **纯乘法专项** — 区别于综合数学 APP，只做乘法，更专业、更聚焦       |
| **目标用户** | 儿童：6-10 岁 / 小学 1-4 年级；家长：需管控学习、看数据              |
| **平台**   | iOS + Android，**手机 + 平板双端精准适配**                       |
| **设计哲学** | 趣味化 · 轻量化 · 分层训练 · 无枯燥刷题 · 零广告 · 护眼             |

### 核心差异化亮点

1. **原理可视化教学** — 先理解后刷题，告别死记硬背（动画演示"几个几相加"）
2. **纯乘法深耕** — 不混加减除法，专注一个知识点做到极致
3. **智能因材施教** — 自动找薄弱点 → 推送专项练习 → 错题清零闭环
4. **家长可控** — 数据透明、时长可管、出题可自定义、纯净模式无干扰
5. **双端适配** — 手机竖屏碎片练习 + 平板横屏大字大键深度学习

## 三、模块一：启蒙入门（零基础专属）

> 目标：理解乘法本质，不死记硬背，轻松掌握九九表

### 3.1 乘法原理可视化教学 ⭐ 核心差异化功能

| 项目         | 说明                                                                 |
| ------------ | -------------------------------------------------------------------- |
| **教学方式**    | 动画 + 图形演示：几个几相加、行列分组计数（如 3×4 = 4 个 3 相加）           |
| **课时划分**    | ① 1~5 乘法 ② 6~9 乘法 ③ 特殊口诀（1/0/10 的乘法）                        |
| **交互要求**    | 支持暂停、回放、快进，儿童自主控制节奏                                     |
| **知识点小结**  | 每课末尾配文字总结（如"乘法就是'几个相同的数相加'的简便运算"）                 |
| **双端适配**    | 平板：全屏高清动画；手机：精简画面保留核心                                   |

**原型对应页面**：`手机-启蒙入门-可视化教学页`（page #20）
- 展示 3×4=12 用圆圈阵列 + 行列分组可视化
- 底部配文字知识点小结

### 3.2 乘法口诀趣味背诵（三种模式）

| 模式       | 功能说明                           | 原型页面                     |
| ---------- | ---------------------------------- | ---------------------------- |
| **跟读背诵**  | 童声朗读 + 单句重复 + 整体背诵 + 完整口诀列表展示 | `手机-跟读背诵页`(#2)、`平板-口诀背诵页`(#10) |
| **遮挡背诵**  | 随机遮挡答案自测记忆，显示已记/复习/待挑战统计      | `手机-遮挡背诵页`(#12)            |
| **口诀小游戏** | 口诀填空、对错判断、匹配连线三种子模式            | `手机-口诀小游戏-填空模式`(#13)       |

**口诀范围选择**：
- ✅ 1~5 乘法（已完成态）/ 🟠 6~9 乘法（进行中）/ ⬜ 特殊口诀（未解锁）

**平板口诀背诵页特色**（#10）：
- 左右分栏布局：左侧当前句大卡片 + 音频控制；右侧完整 9 句口诀列表高亮当前句

### 3.3 零基础入门练习题

| 特性               | 说明                                       |
| ------------------ | ------------------------------------------ |
| **难度**             | 仅限已学口诀出题，不超纲                         |
| **题型**             | 基础口算为主                                 |
| **时间限制**           | ❌ 无时间限制，主打正确率                         |
| **错题反馈**           | 即时弹窗讲解，用图形方式简单释义                    |
| **标签**             | "超简单" — 降低儿童心理门槛                      |

**原型页面**：`手机-入门练习答题页`(#3)
- 大号数字卡片（6 × 4 = ?）+ 点击听题按钮
- 数字键盘输入答案（1-9, 0, 退格, 确定）

## 四、模块二：专项进阶（熟练提升）

> 目标：提升口算速度和准确率，攻克易错点，适配校内考试难度

### 4.1 分层难度体系 L1–L5

| 层级 | 名称     | 内容说明                               | V1.0? |
| ---- | -------- | -------------------------------------- | ----- |
| **L1**  | 基础巩固   | 1~9 单一口诀专项练习（如只练 7 的乘法）          | ✅     |
| **L2**  | 混合基础   | 1~9 随机乘法，无难度倾斜                   | ✅     |
| **L3**  | 提速训练   | 限时口算，训练答题速度                      | ✅     |
| L4   | 易错专攻   | 系统自动抓取高频易错口诀、易混淆题型             | V2.0  |
| L5   | 进阶拔高   | 乘法混合运算、填空、逆向思维、简单应用题           | V3.0  |

**原型页面**：`手机-进阶训练-难度选择页`(#4)
- 卡片式难度选择（L1 已完成 / L2 进行中 / L3 🔒未解锁）

### 4.2 多种训练模式

| 模式       | 说明                  | 适用场景       |
| ---------- | --------------------- | ------------ |
| **自由练习**  | 不限题量、不限时间，随时练随时退 | 日常随意练习    |
| **限时间关**  | 30 秒 / 1 分钟 / 3 分钟限时 | 提升口算速度    |
| **定量刷题**  | 自定义 10/20/50 题，完成打卡 | 有目标训练     |
| **每日打卡**  | 固定 10 题每日基础训练       | 养成习惯       |
| **平板大屏**  | 横屏答题、大字体、大按键       | 平板专项训练    |

**各模式原型页**：

| 页面                              | 内容说明                                           |
| --------------------------------- | -------------------------------------------------- |
| `手机-自由练习答题页`(#14)            | L2 混合基础，比大小题型（7×6 ○ 5×9），选项 > = <        |
| `手机-限时闯关答题页`(#5)             | 计时器 00:42，逆向求数（?×6=48），四选一选项 + 连对激励    |
| `手机-每日打卡页`(#15)               | 日历视图（连续打卡天数）、今日任务 10 题、[开始打卡]        |

### 4.4 答题结果反馈

**原型页面**：`手机-答题结果页`(#6)

- 操作按钮：「再来一次」/ 「继续闯关」

## 五、模块三：错题攻克 & 智能复盘

> 目标：告别盲目刷题，精准补短板

### 5.1 自动错题本

**原型页面**：`手机-错题本列表页`(#7)

| 元素         | 内容                                         |
| ------------ | -------------------------------------------- |
| 头部统计       | 待复习数 / 已掌握数 / 掌握率%（如 12 / 28 / 70%）      |
| 薄弱知识点分析   | 按口诀分类统计（如 7的乘法 5 题、8 的乘法 3 题、逆向求数 4 题） |
| 错题列表       | 每条：题号 + 你的答案 vs 正确答案 + 重做按钮 🔄          |
| 批量操作       | 「全部重做」一键重做所有待复习题目                     |

### 5.2 错题重做 & 智能讲解

**原型页面**：`手机-错题重做页`(#16)

| 元素           | 内容                                                       |
| ------------ | ---------------------------------------------------------- |
| 上次错误回顾      | "上次你写的是 54，正确答案是 56"                                |
| 💡 极简讲解      | 用儿童易懂的生活化语言解释（如"七八五十六——7个8就是56，像一周7天每天8小时睡，一共56小时呢！"） |
| 重做状态追踪      | "第 X 次重做"，记录重做次数                                      |
| 同入口答题体验    | 数字键盘输入，点击听题                                          |

### 5.3 清零机制

- 错题全部重做正确后 → 自动清除该题错题记录
- 记录"已掌握"状态
- 掌握率实时更新
