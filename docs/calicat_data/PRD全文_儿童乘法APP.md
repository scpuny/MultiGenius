## 二、产品概述

### 2.1 产品定位

面向6-10岁小学生（1-4年级），**专注乘法知识点**，从零基础入门到熟练口算、速算进阶，适配手机、平板双设备，主打**趣味化、轻量化、分层训练、无枯燥刷题**，解决孩子乘法记不住、算得慢、容易错、不会灵活运用的问题，替代纸质口算题卡，适配儿童认知节奏，兼顾趣味性与学习专业性。

### 2.2 核心设计原则

* **低门槛入门**：零基础可上手，从理解乘法本质开始
* **阶梯式进阶**：分层难度体系，循序渐进不跳跃
* **游戏化激励**：正向激励驱动，无氪金无诱导
* **护眼适配**：低蓝光、色温调节、时长管控
* **无广告干扰**：纯学习工具，零商业干扰
* **家长可控**：数据透明、时长可管、出题可自定义

### 2.3 核心差异化亮点

* **纯乘法专项深耕**：区别于综合数学APP，只做乘法，更专业、更聚焦
* **原理可视化教学**：先理解后刷题，告别死记硬背
* **手机平板双端精准适配**：针对性设计横/竖屏专属使用场景
* **零干扰纯净学习**：无广告、无引流、无氪金
* **智能因材施教**：系统自动找短板、推专项练习

## 三、目标用户与核心痛点

### 3.1 儿童用户（6-10岁）

| 痛点    | 详细描述                 |
| ----- | -------------------- |
| 刷题枯燥  | 纸质刷题单调，缺乏趣味性         |
| 错题无解  | 错题没人及时讲解，反复错同一类题目    |
| 缺乏针对性 | 没有个性化练习，会的反复做、不会的没练到 |
| 节奏失衡  | 没有口算速度训练，要么太慢、要么粗心出错 |
| 挫败感强  | 基础没学好就做难题，容易抵触数学学习   |

### 3.2 家长用户

| 痛点   | 详细描述                      |
| ---- | ------------------------- |
| 时间不足 | 没时间出题、批改口算作业              |
| 数据缺失 | 无法精准掌握孩子薄弱知识点             |
| 产品混乱 | 市面APP杂糅加减乘除，没有纯乘法专项分层训练   |
| 管控需求 | 担心孩子沉迷、看屏幕伤眼，需可控学习时长和纯净环境 |
| 辅导需求 | 需要清晰学习数据，方便针对性辅导          |

### 3.3 设备适配痛点

* 手机屏幕小，适合碎片化短时练习；平板屏幕大，适合长时间专项训练
* 多数儿童APP未做双端适配，平板界面拉伸、手机按钮过小，孩子操作不便

## 五、核心功能详细设计

### 5.1 乘法启蒙入门模块（零基础专属）

**核心目标**：理解乘法本质，不死记硬背，轻松掌握九九乘法表

#### 5.1.1 乘法原理可视化教学（核心差异化功能）

| 功能点  | 详细说明                                    |
| ---- | --------------------------------------- |
| 教学方式 | 动画+图形演示乘法逻辑：几个几相加、行列分组计数（如3×4=4个3相加）    |
| 平板适配 | 大屏高清动画演示，全屏展现视觉细节                       |
| 手机适配 | 精简画面，保留核心知识点，适合小屏观看                     |
| 课时划分 | ① 1-5乘法 ② 6-9乘法 ③ 特殊口诀（1的乘法、0的乘法、10的乘法） |
| 交互要求 | 支持暂停、回放、快进，儿童可自主控制学习节奏                  |

#### 5.1.2 乘法口诀趣味背诵

| 模式    | 功能说明                           |
| ----- | ------------------------------ |
| 跟读背诵  | 标准童声朗读，逐句跟读，全屏口诀展示，支持单句重复、整体背诵 |
| 遮挡背诵  | 随机遮挡口诀答案，自测记忆程度                |
| 口诀小游戏 | 口诀填空、对错判断、匹配连线，替代枯燥背诵          |

#### 5.1.3 零基础入门练习题

* 难度最低，仅限已学口诀出题，不超纲
* 题型以基础口算为主
* **无时间限制**，主打正确率，保护孩子学习信心
* 错题即时弹窗讲解，用图形方式简单释义，告诉孩子错因

***

### 5.2 乘法进阶训练模块（熟练提升、专项突破）

**核心目标**：提升口算速度、准确率，攻克易错点，适配校内考试难度

#### 5.2.1 分层难度体系（L1-L5）

| 层级 | 名称   | 内容说明                             |
| -- | ---- | -------------------------------- |
| L1 | 基础巩固 | 1-9单一口诀专项练习（如只练7的乘法）             |
| L2 | 混合基础 | 1-9随机乘法，无难度倾斜                    |
| L3 | 提速训练 | 限时口算，训练答题速度                      |
| L4 | 易错专攻 | 系统自动抓取高频易错口诀、易混淆题型               |
| L5 | 进阶拔高 | 乘法混合简单运算、填空、逆向思维题（（）×6=48）、简单应用题 |

#### 5.2.2 多种训练模式

| 模式     | 说明                   | 适用场景   |
| ------ | -------------------- | ------ |
| 自由练习   | 不限题量、不限时间，随时练随时退     | 日常随意练习 |
| 限时闯关   | 30秒/1分钟/3分钟限时答题，挑战速度 | 提升口算速度 |
| 定量刷题   | 自定义10/20/50题，完成打卡    | 有目标训练  |
| 每日打卡   | 固定10题每日基础训练，养成学习习惯   | 每日习惯养成 |
| 平板大屏模式 | 横屏答题、大字体、大按键，适合远距离操作 | 平板专项训练 |

#### 5.2.3 题型全覆盖

| 题型      | 示例                |
| ------- | ----------------- |
| 基础口算    | 7×8=？             |
| 口诀填空    | 六八\_\_？           |
| 判断对错    | 5×6=30 ✅ / ❌      |
| 逆向求数    | （）×6=48           |
| 比大小     | 7×6 ○ 5×9         |
| 简单图文应用题 | 每排4棵树，种了3排，一共多少棵？ |

***

### 5.3 错题攻克&智能复盘模块

**核心目标**：告别盲目刷题，精准补短板

| 功能      | 详细说明                           |
| ------- | ------------------------------ |
| 自动错题本   | 所有错题自动收录，标注错误原因、对应口诀知识点，支持随时重做 |
| 薄弱点智能推送 | 系统统计常错的乘法口诀、题型，自动生成专属专项练习      |
| 错题复盘讲解  | 每道错题配套极简儿童易懂讲解，不堆砌复杂公式         |
| 清零机制    | 错题全部重做正确后，自动清除错题记录，记录掌握情况      |

***

### 5.4 儿童激励体系

**设计原则**：针对儿童天性设计，无氪金、无诱导消费，纯正向激励

| 激励方式   | 说明                                     |
| ------ | -------------------------------------- |
| 闯关星级奖励 | 根据正确率、答题速度给予1-3星评价                     |
| 积分勋章系统 | 每日打卡、闯关成功、错题清零获得积分，解锁专属勋章（口算小能手、口诀大师等） |
| 成长进度条  | 可视化展示口诀掌握进度、刷题总量、正确率提升曲线               |
| 趣味彩蛋   | 连续打卡、满分闯关触发轻量动画奖励，无广告、无弹窗干扰            |

***

### 5.5 家长管理&数据中心模块

#### 5.5.1 实时学习数据

展示每日学习时长、做题数量、正确率、薄弱知识点、错题汇总

#### 5.5.2 成长报告

* 日报告：今日学习概况
* 周报告：本周进步趋势
* 月报告：月度学习总结 + 针对性提升建议

#### 5.5.3 设备&时长管控（核心刚需）

| 功能     | 说明                     |
| ------ | ---------------------- |
| 每日时长限制 | 家长自定义每日最大学习时长，超时自动锁屏   |
| 禁用时间段  | 禁止夜间使用，设置禁用时间段，保护视力和作息 |
| 纯净模式   | 屏蔽所有弹窗、广告、跳转链接         |

#### 5.5.4 自定义出题

家长可手动指定练习口诀范围、出题数量、难度等级，针对性辅导孩子短板

***

### 5.6 双端设备适配专项功能

#### 平板适配（横屏优先）

* 超大字体、超大点击按键
* 界面布局宽松，适配儿童粗手指操作
* 全屏动画教学，视觉体验更好
* 适合长时间学习

#### 手机适配（竖屏优先）

* 精简界面、轻量化题型
* 适合碎片化短时练习
* 单手可操作

#### 通用护眼模式

* 冷暖色温调节
* 低蓝光模式
* 夜间护眼模式
* 适配儿童视力保护需求

#### 触摸操作适配

* 全部功能点击、滑动操作
* 无复杂输入
* 低龄儿童可独立使用

***

### 5.7 基础体验功能（儿童APP必备）

* ❌ 无开屏广告、无信息流、无游戏引流、无付费诱导
* ✅ 纯学习工具属性
* ✅ 支持离线使用，下载后无需联网即可刷题、看错题
* ✅ 童声语音播报题目，认字少的低龄儿童也能独立做题
* ✅ 操作极简，3步内进入刷题界面，无复杂层级

## 七、技术要点与非功能需求

### 7.1 性能要求

| 指标    | 要求             |
| ----- | -------------- |
| 启动速度  | 冷启动 ≤ 2秒       |
| 答题响应  | 点击到下一题 ≤ 300ms |
| 动画流畅度 | ≥ 60fps（教学动画）  |
| 离线可用性 | 核心功能100%离线可用   |

### 7.2 安全与隐私

* 儿童个人信息保护合规（符合《儿童个人信息网络保护规定》）
* 不收集非必要个人信息
* 数据传输加密
* 无第三方SDK广告追踪

### 7.3 兼容性

* iOS 13+ / Android 8+
* 手机分辨率适配：320pt - 430pt 宽度
* 平板分辨率适配：横屏优先，支持分屏模式
