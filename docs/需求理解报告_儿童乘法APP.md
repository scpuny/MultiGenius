## 三、四大核心模块功能详解

### 3.1 启蒙入门（零基础专属）
- **可视化教学**：把乘法解释为「几个相同的数相加」，如 `3×4 = 4个3相加 = 4行×3列=12`，配圆点网格动画。分课时：1-5乘法 / 6-9乘法 / 特殊口诀（0、1、10）。
- **口诀背诵三方式**：
  - *跟读背诵*：标准童声朗读、逐句跟读、全屏口诀、单句重复/整体背诵；
  - *遮挡背诵*：随机遮挡答案自测，记录「已记住/需复习/待挑战」；
  - *口诀小游戏*：口诀填空、对错判断、匹配连线，替代枯燥背诵（含生命值 ❤️ 机制）。
- **入门练习**：难度最低，仅限已学口诀、不限时、主打正确率，保护学习信心。

### 3.2 专项进阶（熟练提升）
- **分层难度 L1–L5**：L1 单一口诀专项 → L2 随机混合 → L3 限时提速 → L4 易错专攻 → L5 进阶拔高。原型难度选择页展示 L1(已完成)/L2(进行中)/L3(🔒 待解锁)。
- **多种训练模式**：自由练习（不限时）、限时闯关（30秒/1分/3分）、定量刷题（10/20/50题）、每日打卡（固定10题）。
- **题型覆盖**：基础口算、口诀填空、判断对错、逆向求数（?×6=48）、比大小（7×6 ○ 5×9）、简单图文应用题。

### 3.3 错题攻克 & 智能复盘
- **自动错题本**：所有错题自动收录，标注「你的答案/正确答案」。
- **薄弱点智能推送**：按口诀/题型统计高频错误（如 7的乘法/8的乘法/逆向求数）。
- **错题复盘讲解**：每道错题配极简儿童化记忆提示（如「七八五十六，7个8就是56」）。
- **清零机制**：重做正确后清除记录。

### 3.4 儿童激励体系（无氪金）
- 闯关星级奖励（1–3★）、积分勋章（口算小能手/打卡达人/全对之星/口诀大师）、成长进度条、连续打卡/满分闯关 趣味彩蛋。

### 3.5 家长管理 & 数据中心
- **实时数据**：每日时长、做题数、正确率、薄弱点、错题汇总。
- **成长报告**：日报/周报/月报 + 综合表现评分 + 针对性提升建议。
- **设备&时长管控**：每日时长限制（默认 30 分钟）、禁用时间段（默认 21:00–07:00）、纯净模式（屏蔽弹窗/广告）。
- **自定义出题**：家长选口诀范围、题量、难度、题型，生成专属练习。

### 3.6 双端适配 & 护眼
- 手机：竖屏、精简界面、碎片化短时练习；平板：横屏、超大字体按键、双栏布局、长时间专项训练。
- 护眼：护眼模式、冷暖色温调节、低蓝光模式、夜间护眼（21:00–07:00 自动暖色）、亮度调节。

## 四、关键用户流程（原型印证）

1. **首次进入**：启动闪屏 → 学习广场首页（进度看板 + 4 模块入口 + 今日推荐）。
2. **零基础入门**：首页「启蒙入门」→ 选口诀段（1-5/6-9/特殊）→ 选背诵方式（跟读/遮挡/小游戏）→ 入门练习。
3. **进阶刷题**：首页「专项进阶」→ 选难度(L1-L3) + 模式(自由/限时/定量/每日) → 答题（口算/逆向/比大小等）→ 答题结果（星级/积分/错题收录）。
4. **错题复盘**：首页「错题本」→ 查看薄弱点 → 全部重做/单题重做（带记忆提示）→ 清零。
5. **家长管控**：「我的」→ 个人中心（数据看板）→ 成长报告 / 自定义出题 / 护眼设置 / 时长限制。

## 六、设计要点观察

- **导航结构清晰**：底部 4 Tab（首页/训练/错题本/我的）贯穿全端，模块边界明确。
- **儿童友好交互**：童声听题（📢 点击听题）、数字键盘作答、大圆角卡片、正向激励文案（「太棒了！」「连对5题！」）。
- **家长/儿童双视角**：儿童侧重趣味与激励，家长侧重数据与管控，通过「个人中心」汇聚。
- **状态可视化**：进度条、连续打卡天数、掌握率、勋章墙，强化成就感。
- **护眼贯穿**：护眼模式为独立设置页，并与夜间禁用时段联动。
