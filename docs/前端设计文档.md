# 乘法小天才 · 前端设计文档

> 依据：Calicat 设计文件 `2077760032043438080`（画布 1：22 页 V1/V2；画布 2：6 页 V3）
> 技术栈：**Tauri 2（Rust 后端）+ React 18 + Vite 5 + TypeScript 5（前端）**
> 通信方式：**Tauri 桥接命令（Command Bridge）**——前端通过 `invoke` 调用 Rust 后端命令
> 适用对象：前端 / 客户端开发、架构评审

---

## 1. 整体架构

```
┌──────────────────────────────────────────────────────────┐
│                      Tauri Application                      │
│                                                            │
│  ┌──────────────────────┐      ┌──────────────────────┐   │
│  │   Frontend (WebView)  │      │   Backend (Rust)      │   │
│  │   React + Vite + TS   │      │   Tauri 2 + Commands  │   │
│  │                       │      │                       │   │
│  │  Pages / Components   │ ──invoke──▶ │  #[tauri::command] │   │
│  │  Stores (Zustand)     │ ◀─Result/─│  Serde 序列化        │   │
│  │  Services(bridge.ts)  │   Event   │  SQLite 本地存储     │   │
│  │  Theme(tokens.css)    │          │  音频/文件/系统 API   │   │
│  └──────────────────────┘      └──────────────────────┘   │
│          ▲                           │                      │
│          └────── Tauri Window ───────┘                      │
└──────────────────────────────────────────────────────────┘
```

**核心原则**
- **离线优先**：核心功能 100% 离线可用（PRD 7.1），所有数据存本地 SQLite，无服务端依赖。
- **桥接命令**：前端不直接碰存储/系统，统一通过 `invoke('command_name', args)` 调 Rust 命令；Rust 命令返回 `Result<T, String>`。
- **双端同源**：手机（390 宽）/ 平板（1024 宽）共用同一套组件，靠 CSS 断点 + 容器查询适配，非两套代码。
- **多账号隔离**：所有读写命令携带 `child_id`，后端按 child 隔离进度/积分/错题/报告。

## 2. 技术选型

| 层 | 选型 | 理由 |
| --- | --- | --- |
| 构建 | Vite 5 | 快冷启、HMR；Tauri 官方推荐 |
| 语言 | TypeScript 5（strict） | 类型安全，桥接命令前后端共享模型 |
| UI | React 18 + React Router 6 | SPA 路由，28 页映射 |
| 状态 | Zustand | 轻量、无 Provider 嵌套；适合 child/settings 全局态 |
| 样式 | CSS Modules + 设计 Token（CSS 变量） | 高度定制化童趣 UI，避免 UI 库覆盖；token 复用原型配色 |
| 后端 | Rust + Tauri 2 | 小体积、安全；命令即 API |
| 存储 | SQLite（`rusqlite` 或 `sqlx`） | 本地离线、结构化查询 |
| 音频 | Tauri + Web Audio / `tauri-plugin` | 童声听题（📢）、语音讲解 |
| 导出 | `tauri-plugin-dialog` + 文件系统 | 报告导出 PNG/PDF、分享 |

## 4. 数据模型（前后端共享，`src/types/models.ts`）

```ts
// 多账号
export interface Child {
  id: string;            // child_id（UUID）
  name: string;
  avatar: string;        // emoji 或色值
  grade: 1 | 2 | 3 | 4;  // 年级
  points: number;        // 积分
  accuracy: number;      // 正确率 %（如 83）
  createdAt: number;
}
export const MAX_CHILDREN = 4;

// 进度
export interface Progress {
  childId: string;
  masteredFormulas: number;   // 已掌握口诀数（如 18）
  totalFormulas: number;      // 总口诀数（45）
  streakDays: number;         // 连续打卡
  todayQuestions: number;
  todayMinutes: number;
  todayAccuracy: number;
}

// 题型
export type QuestionType =
  | 'basic'        // 基础口算 7×8=?
  | 'fill_blank'   // 口诀填空 六八__？
  | 'judge'        // 判断对错
  | 'reverse'      // 逆向求数 ?×6=48
  | 'compare'      // 比大小 7×6 ○ 5×9
  | 'word_problem';// 图文应用题

export type Difficulty = 'L1' | 'L2' | 'L3' | 'L4' | 'L5';
export type Module = 'enlightenment' | 'advanced' | 'wrong_review' | 'application';
export type PracticeMode = 'free' | 'timed' | 'quantitative' | 'daily';

export interface Question {
  id: string;
  type: QuestionType;
  expression: string;     // "7×8"
  answer: number;         // 56
  options?: number[];     // 选择题选项
  wordProblem?: { text: string; image?: string; hint?: string };
  difficulty: Difficulty;
  module: Module;
}

// 答题会话
export interface SessionResult {
  childId: string;
  module: Module;
  mode: PracticeMode;
  difficulty: Difficulty;
  questionCount: number;
  correctCount: number;
  durationSec: number;
  wrongQuestionIds: string[];
}
export interface SessionSummary {
  correctRate: number;     // %
  stars: 1 | 2 | 3;
  gainedPoints: number;
  newWrong: { expression: string; userAnswer: number; correctAnswer: number }[];
}

// 错题
export interface WrongQuestion {
  id: string;
  childId: string;
  question: Question;
  userAnswer: number;
  reviewCount: number;
  status: 'pending' | 'mastered';
  tip: string;            // 生活化记忆提示
}

// 报告
export type ReportPeriod = 'day' | 'week' | 'month';
export interface Report {
  childId: string;
  period: ReportPeriod;
  date: string;           // 2025-01-08
  durationMin: number;
  questionCount: number;
  accuracy: number;
  newWrong: number;
  score: number;          // 综合表现分
  weakPoints: { name: string; percent: number }[];
  suggestions: string[];
}

// 设置（护眼 + 时长）
export interface RestReminder { enabled: boolean; intervalMin: number; restMin: number; }
export interface DisabledPeriod { name: string; start: string; end: string; days?: string; }
export interface Settings {
  childId: string;
  dailyLimitMin: number;          // 每日总时长上限
  moduleLimits: Record<string, number>; // 分模块上限（0=不限）
  restReminder: RestReminder;
  disabledPeriods: DisabledPeriod[];
  eyeCare: { blueLightFilter: boolean; nightMode: boolean; colorTemp: 'warm' | 'natural' | 'cool'; brightness: number };
}
```

## 5. Tauri 桥接命令 API 设计（核心）

> 前端统一经 `services/bridge.ts` 封装调用；Rust 端在 `commands.rs` 实现。
> 约定：所有命令返回 `Result<T, String>`；多账号命令必须带 `child_id`；错误用中文描述。

### 5.1 账户管理（多账号）
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `get_children` | — | `Child[]` | 列出全部孩子（最多 4） |
| `add_child` | `{name, avatar, grade}` | `Child` | 新增（超 4 个报错） |
| `update_child` | `{id, name?, avatar?, grade?}` | `Child` | 编辑 |
| `delete_child` | `{id}` | `void` | 删除（连带数据） |
| `switch_child` | `{id}` | `void` | 切换当前孩子（前端 store 同步） |

### 5.2 进度
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `get_progress` | `{childId}` | `Progress` | 当前进度看板 |
| `update_progress` | `{childId, deltaQuestions, deltaMinutes, deltaCorrect}` | `Progress` | 答题后更新 |

### 5.3 练习 / 答题
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `generate_questions` | `{module, difficulty, mode, count, formulaRange?}` | `Question[]` | 生成题目（按难度/题型） |
| `submit_session` | `SessionResult` | `SessionSummary` | 提交会话→算星/积分/收录错题 |
| `get_session_result` | `{sessionId}` | `SessionSummary` | 取结果（结果页用） |

### 5.4 错题攻克
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `list_wrong_questions` | `{childId, filter?}` | `WrongQuestion[]` | 错题库（含薄弱点统计） |
| `review_wrong_question` | `{childId, wrongId, userAnswer}` | `{correct, tip, status}` | 重做单题 |
| `clear_wrong_question` | `{childId, wrongId}` | `void` | 清零（做对后） |
| `batch_review` | `{childId}` | `WrongQuestion[]` | 全部重做队列 |

### 5.5 报告 / 导出
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `get_report` | `{childId, period, date}` | `Report` | 日/周/月报 |
| `export_report` | `{childId, period, format:'png'|'pdf'}` | `{path}` | 生成文件并落盘 |

### 5.6 设置 / 护眼 / 时长
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `get_settings` | `{childId}` | `Settings` | 读设置 |
| `update_settings` | `Settings` | `Settings` | 存设置 |

### 5.7 应用题专项（V3）
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `generate_application` | `{dimensions:QuestionType[], difficulty, count}` | `Question[]` | 图文应用题（带 wordProblem） |

### 5.8 自定义出题（V3 家长）
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `generate_custom_quiz` | `{childId, formulaRange, count, difficulty, types:QuestionType[]}` | `Question[]` | 家长自定义 |

### 5.9 打卡
| 命令 | 入参 | 返回 | 说明 |
| --- | --- | --- | --- |
| `get_checkin` | `{childId, month}` | `{streak, monthCount, total, doneDays:number[]}` | 日历数据 |
| `do_checkin` | `{childId}` | `void` | 完成今日 10 题打卡 |

> **事件（后端→前端，可选）**：`practice_completed`（答题完成弹窗）、`time_limit_reached`（时长到锁屏）、`rest_reminder`（休息提醒）。前端用 `listen` 订阅。

## 7. 状态管理（Zustand）

- `useChildStore`：`children: Child[]`、`currentChildId`、`switchChild()`（调 `switch_child` 并刷新）、`loadChildren()`。
- `useSettingsStore`：`settings: Settings`、`load()`、`save()`（调 `update_settings`）。
- `usePracticeStore`：当前 `SessionResult` 草稿、计时器、`submit()`。
- 全局注入 `childId`：所有 bridge 调用自动带 `currentChildId`（bridge.ts 内部读取 store）。

## 9. 非功能要点（来自 PRD 7）

- 冷启动 ≤ 2s；点击下一题 ≤ 300ms；动画 ≥ 60fps。
- 离线 100%：SQLite 本地，无网络依赖。
- 儿童隐私合规：不收集非必要信息，本地加密。
- 无广告/无第三方 SDK 追踪。

## 10. 打包与构建（Tauri）

`tauri.conf.json` 要点：
```jsonc
{
  "productName": "乘法小天才",
  "identifier": "com.chengfa.xiaotiantian",
  "build": { "frontendDist": "../dist", "devUrl": "http://localhost:5173" },

      "title": "乘法小天才",

    "security": { "csp": "default-src 'self'" }
```

- 开发：`npm run tauri dev`；构建：`npm run tauri build`。
- `capabilities/default.json` 需声明允许前端的命令（如 `get_children`、`submit_session`…）与 `core:event:listen`。
- 图标：`src-tauri/icons/`（需 32/128/icon.icns/png）。

## 11. 开发里程碑建议

1. **M1 骨架**：配置 + 路由 + BottomNav + Token + 首页（打通 invoke 闭环）。
2. **M2 练习闭环**：generate_questions → 答题页 → submit_session → 结果页 → 错题收录。
3. **M3 启蒙 + 错题**：口诀三方式、可视化教学、错题本/重做/精准讲解。
4. **M4 家长 + 设置**：报告、自定义出题、护眼、时长管控、多账号切换。
5. **M5 V3 应用题 + 导出 + 平板适配 + 打包上架**。
